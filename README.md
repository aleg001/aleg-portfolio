# 🕸 Web Portfolio - Ale Gómez 🕸

Hosteado en: https://portfolio-aleg001.web.app/

## 🦾  Tecnologías implementadas  🦾

- VueJs - 2.6.14
- Vuetify - 2.6
- Firebase hosting
- Babel — 7.12.16 
- ESlint - 7.32
- SASS - 1.32
- CSS 



## Configuración de proyecto

### Iniciar proyecto 🧑🏽‍💻

```
npm install
```

### Compilar y hot reload 🧑🏽‍💻

```
npm run serve
```

### Producción 🧑🏽‍💻

```
npm run build
```
### Firebase Deployment 🧑🏽‍💻

```
firebase deploy
```


### eslint 🧑🏽‍💻

```
npm run lint
```
